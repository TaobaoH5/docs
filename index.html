开发约定

一、文档

文档的规范遵循JSDoc。一些细节的要求

  • 1、文件开头第一行，须写整个文件的注释，格式如下：

    /**
* @fileOverview This is a HelloWorld!
* @author zhuxun
* @version 0.5
* @requires debug@0.5
*/

  • 2、文档注释必须紧跟下面的代码，不出现空行，且以下代码必须有注释：构造函数（要写清楚继承或实现自某个类或接口）、可访问变量或重要变量、成员方法（要写清楚访问修饰符，以及是否覆盖了父类的方法）、类方法、以及方法体内的this上下文是具体指向的什么。

  • 3、不再采用的方法，标注为@deprecated。

  • 4、解释重要业务逻辑，用单行注释。需要后期关注或增加的功能，采用// TODO注释。

  • 5、具体请查看这两份文档：http://code.google.com/p/jsdoc-toolkit/wiki/TagReference，https://developers.google.com/closure/compiler/docs/js-for-compiler#tags。

二、模块信息和依赖相关

每个模块（独立一个文件夹，以模块名命名）都包含一个模块主文件（即模块名.js），相同目录下有一个package.json，储存了该模块的基本信息，格式如下：

{
    "name" : "helloworld",
    "description" : "say hello world",
    "author" : "zhuxun",
    "version" : "0.5",
    "namespace" : "mix/norjs/test",
	"main" : "./helloworld",
    "dependencies" : {
        "mix/norjs/test/debug" : "0.5"
    },
    "stable-release" : ["0.1"] // 此项为历史稳定版本列表
}

三、模块代码

遵循CommonJS，不基于CMD、AMD，XMD等来编写。例如helloworld.js：

/**
* @fileOverview This is a HelloWorld!
* @author zhuxun
* @version 0.5
* @requires debug@0.5
*/

var Debug = require('debug'),
	debug = new Debug();
/**
* HelloWorld
* @constructor
*/
function HelloWorld() {
    /** @type {string} */
    this.word = 'helloworld';
    debug.log('create helloworld');
}

/**
 * say the word
 * @param {string=} word
 * @this {HelloWorld}
 */
HelloWorld.prototype.say = function(word) {
    return word || this.word;
}

module.exports = HelloWorld;

四、模块Build

目前对于模块代码的build，可以写两套方案，一套是针对seajs的，一套是直接调用的。 build生成的文件位于模块目录的build文件夹下，文件命名规则可配置，默认为模块名-v版本号-build器.js，例如hellworld-v0.1-seajs.js。

  • 1、seajs-build： 

    /**
* @fileOverview This is a HelloWorld!
* @author zhuxun
* @version 0.5
* @requires debug@0.5
*/

define(
'mix/norjs/test/helloworld@0.5', // resolve id from package.js when build
['mix/norjs/test/debug@0.5'], // resolve deps from package.json when build
function(require, exports, module) {
    // 原helloworld.js的代码
});

  • 2、normal-build： 

    /**
* @fileOverview This is a HelloWorld!
* @author zhuxun
* @version 0.5
* @requires debug@0.5
*/

/*===generated by norjs build===*/
(function(require, exports, module){

/*===source code start here===*/
// 原helloworld.js的代码

/*===generated by norjs build===*/
}).apply(
window,
norjs(
    "mix/norjs/test/helloworld@0.5"
    ,["mix/norjs/test/debug@0.5"]
));
