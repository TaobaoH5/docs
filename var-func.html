变量

背景色

@standard-bg-color #EEE 主要背景色
@standard-main-color #F40 主色
@standard-sub-color1 #F60 辅色A
@standard-sub-color2 #F90 辅色B
@standard-linear-l #F94B21 渐变色开始
@standard-linear-r #EB3E12 渐变色结束

文字属性

@text-main-color #333 主色 TEXT
@text-highlight-color #F40 高亮 TEXT
@text-sub-color #666 辅色 TEXT
@text-em-color #D40000 着重色 TEXT
@text-link-color #047 链接色 TEXT
@title-font-size 24px 标题文字大小
@tab-font-size 18px 标签文字大小
@big-font-size 16px 大号文字
@normal-font-size 14px 中号文字
@small-font-size 12px 小号文字

间距

@page-padding-width 10px 页边距
@list-margin-width 30px 列表间距
@first-list-margin-width 20px 第一个列表的顶部间距
@list-item-margin-width 10px 列表中各项目间距
@group-list-margin-width 20px 分组列表间距

屏幕尺寸

@screen-min-width 320px 最小宽度
@screen-min-height 480px 最小高度

函数

关于函数

简单函数

.element {
  .clearfix();
}

带参函数

.element {
  .border-radius(4px);
}

使用函数

工具

函数名 参数 用途
.clearfix() none 清除浮动
.none() none 隐藏元素display:none
.no-spacing() none 清除元素边距
.margin-center() none 元素和文字居中

CSS3 属性

函数名 参数 用途
.border-radius() @x 设置圆角
.transform() @x 设置CSS3的变换
.translate() @x:0, @y:0, @origin:0 0, @duration:0s 设置CSS3的平移
.animation() @name, @duration, @func:linear, @count:1 设置CSS3的动画
.rotate() @degrees 旋转元素
.box-shadow() @r, @b, @x, @y, @c 设置元素阴影
.solid-arrow() @width, @height, @color, @bgcolor:transparent, @rotate:0deg 描绘实体箭头
.border-arrow() @width, @height, @color, @border-width:1px, @rotate:0deg 描绘边框箭头
.btn-with-arrow() @w, @h, @aw, @b:1px 描绘带有左箭头的按钮，DOM结构： 
<div>
    <span><em></em></span>
    <a>带箭头的按钮</a>
</div>
.stretch-side() @background, @space-width:9999px 延伸元素背景

编译

注意: 请勿随意改动和覆盖组件库里的样式

可以用作编译的工具

命令行

通过Node安装LESS命令行工具，并运行如下命令：

$ lessc component.less > component.css

使用 --compress 命令参数可以对CSS文件进行压缩

Javascript

下载最新的Less.js 在 <head>引入这个JS

<link rel="stylesheet/less" href="/path/to/component.less">
<script src="/path/to/less.js"></script>

详细功能请参见Wiki内的教程
